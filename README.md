# Fata's Car Kit 

A full-stack car accessories e-commerce website.

## Features
- Product listing
- Product details page
- Shopping cart
- User registration & login
- Checkout & order system

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: MongoDB

## How to run

### Backend
cd backend  
npm install  
npm run dev  

### Frontend
npm install  
npm run dev
